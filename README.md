##### Installing PoshSec
1. Download module
2. Unblock .zip file
3. Extract module files to $env:userprofile\Documents\WindowsPowerShell\Modules
4. Launch PowerShell
5. Import Module
```PowerShell
Import-Module -Name PoshSec -Force
```
